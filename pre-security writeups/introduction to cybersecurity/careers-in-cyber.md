## Summary

Exploration of the cybersecurity career landscape. Maps out the major roles, what each does day-to-day, and what skills and certifications are relevant to each path — helping narrow down a specialization.

---

## Key Concepts Learned

- **Penetration tester** — Ethically attacks systems to find vulnerabilities before real attackers do; requires deep technical and creative thinking
- **SOC analyst** — Monitors security tools, triages alerts, and responds to incidents in real time; the most common entry point into defensive security
- **Red teamer** — Simulates full adversary campaigns to test an entire organization's defense capability
- **Malware analyst** — Reverse-engineers malicious software; sits at the intersection of programming, security, and forensics
- **Threat intelligence analyst** — Researches threat actors and attack trends to help defenders stay ahead
- **Security engineer** — Builds and maintains security infrastructure (firewalls, SIEM, detection rules) that everyone else relies on

## Notes & Walkthrough

### Task 1 — Security Analyst

SOC analysts are responsible for monitoring and protecting networks. They work with logs, alerts, and SIEM tools to triage and investigate threats.

Key skills: log analysis, SIEM, network monitoring, incident triage, communication

### Task 2 — Security Engineer

Engineers design and implement security solutions. They build the systems analysts use every day.

Key skills: networking, system administration, scripting, security architecture

### Task 3 — Incident Responder

Incident responders react when a breach is confirmed. They contain damage, collect evidence, and produce reports.

Key skills: DFIR, forensics, malware analysis, report writing — **directly aligned with my target role**

### Task 4 — Digital Forensics Examiner

Forensics examiners analyze digital evidence for legal and investigative purposes. Work includes disk imaging, memory analysis, and chain of custody.

### Task 5 — Malware Analyst

Analysts dissect malicious software to understand its behavior, origin, and impact. Combines reverse engineering with threat intelligence.

### Task 6 — Penetration Tester

Pen testers simulate attacks on systems with permission to find exploitable weaknesses before real attackers do.

### Task 7 — Red Teamer

More advanced than pen testing — simulates nation-state level adversaries to test full organizational defenses.
